# ERP Suite

This repository contains the MVP ERP as a modular monolith:

- Backend: ASP.NET Core + EF Core + PostgreSQL
- Frontend: React + shadcn/ui
- Architecture: Clean Architecture with module boundaries

## Top-level Structure

- `backend/`: API host, modules, and tests
- `frontend/`: React SPA and shared UI foundations
- `docs/`: PRD and architecture guardrails
- `infra/`: docker and scripts for local/devops setup

Refer to `docs/PRD-ERP-MVP.md` and `docs/MVP-Architecture-Guardrails.md` for product and architecture guidance.
